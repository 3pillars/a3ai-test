# Apple M5 Chip Architecture

Source: Tech summary

## What's New
- M5 Pro and Max: 2 chips fused together
- CPU and GPU on separate silicon pieces
- Reconnected using data center AI chip packaging tech

## The Innovation
- Same CPU in Pro and Max (identical)
- Difference: GPU only
  - Pro: 20 cores
  - Max: 40 cores

## Why It Matters
1. Smaller chips = cheaper to make
2. Run cooler under heavy loads
3. **Scale GPU up/down without redesigning whole chip**
4. Foundation for next decade of Apple Silicon
5. Path to Ultra and beyond

## Significance
- Most important chip design change in Apple Silicon history
- "Lego blocks for silicon"
- Enables faster iteration on GPU without redesigning CPU
