# Research: Google Stitch

**Date:** March 15, 2026
**Source:** Web Search & Article Analysis

## What is Google Stitch?

AI-powered UI design tool from Google Labs (experimental)
- Launches: Google I/O 2025
- Website: stitch.withgoogle.com

## Core Features

1. **Text → UI** — Describe in plain English, get HTML/CSS/React
2. **Image → UI** — Upload sketches/wireframes
3. **Iterative Design** — Branch off, compare versions
4. **Multi-select** — Apply changes to multiple screens
5. **Export** — HTML/CSS/React or Figma

## Models

| Mode | Model | Use Case |
|------|-------|----------|
| Experimental | Gemini 2.5 Pro | Complex, image input |
| Standard | Gemini 2.5 Flash | Quick, Figma export |

## Limits (Free)

- 350 standard screens/month
- 200 pro screens/month

## Pros

- Fast UI generation
- Easy to use
- Free tier generous
- Gemini-powered

## Cons

- No Figma export in Experimental mode
- Inconsistent alignment
- Still needs human refinement

## Use Cases

- Rapid prototyping
- MVP mockups
- Design exploration
- Turn ideas into code
